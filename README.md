# 🧜‍♀️ Mermaid Diagram Builder - Little Mermaid Edition

A fun and interactive Mermaid diagram builder that uses Little Mermaid characters instead of traditional shapes!

## ✨ Features

- **🎨 Character-Based Shapes**: Use Little Mermaid characters as diagram elements
  - 🧜‍♀️ **Ariel** - Start/End nodes
  - 👑 **King Triton** - Decision/Process nodes
  - 🦀 **Sebastian** - Data/Database nodes
  - 🐠 **Flounder** - Action/Process nodes
  - 🧙 **Ursula** - Alternative/Branch nodes
  - 🤴 **Prince Eric** - Input/Output nodes
  - 🏰 **Castle** - External Systems

- **🎯 Interactive Diagram Builder**: Drag-and-drop interface to create diagrams
- **📊 Real-time Preview**: See your Mermaid diagram update instantly
- **📚 Character Legend**: Learn what each character represents
- **🎨 Tailwind CSS Styling**: Beautiful, modern UI with ocean theme
- **💾 Export Code**: Get the Mermaid code for your diagrams
- **📱 Responsive Design**: Works on desktop, tablet, and mobile
